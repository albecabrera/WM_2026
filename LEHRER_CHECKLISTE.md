# Checkliste für Lehrkräfte — WM 2026 Tipp-Spiel

> Diese Liste vor der Präsentation an die Klasse durchlesen.

---

## 1. Datenschutz zuerst

- ✅ **Nur Phantasienamen verwenden** (z. B. „Rasenrakete"). Niemals echte Vor- oder Nachnamen, E-Mail-Adressen oder Geburtsdaten eingeben.
- ✅ Die Zuordnung „welcher Code gehört zu welchem Kind" bleibt **auf Papier bei der Lehrkraft** — nicht in der App, nicht im Klassenchat.

---

## 2. Login richtig erklären (2 Schritte)

**Schritt 1 — Schule wählen und Schulcode eingeben:**

| Schule | Schulcode |
|--------|-----------|
| BBG    | `bbg-wm2026` |
| ESG    | `esg-wm2026` |

**Schritt 2 — Persönlichen Code eingeben** (z. B. `sturmheld-k1`).

> ⚠️ Jeder Code ist einmalig. Jedes Kind benutzt **nur seinen eigenen Code** — nicht den der anderen.

---

## 3. Codes vorher ausdrucken

1. Admin-Bereich öffnen (`admin2026`).
2. **Klassenliste** auswählen → eigene Klasse wählen → **Drucken**.
3. Der Ausdruck enthält zwei Bereiche:
   - **Übersichtsliste** — alle Schüler mit Codes (für die Lehrkraft)
   - **Ausschneidbare Karten** (✂) — eine Karte pro Schüler mit Leerzeile für den echten Namen, Phantasiename und Login-Code
4. Karten ausschneiden, echten Namen draufschreiben und verteilen — **vor der Stunde**.

> Tipp: QR-Codes pro Login-Code sind ebenfalls im Admin-Bereich verfügbar. Ausdrucken und aushängen spart Tippfehler.

---

## 4. Tippen — die wichtigsten Regeln für die Klasse

- 🔒 Ein Spiel kann nur **bis 5 Minuten vor Anpfiff** getippt werden — danach gesperrt.
- 👉 Den Kindern sagen: **früh tippen**, nicht erst kurz vor dem Spiel.
- 🏆 Der **Weltmeister-Tipp** ist nur **bis Turnierbeginn (11. Juni 2026)** möglich — danach nicht mehr änderbar.

---

## 5. Punkte einfach erklären

| Ergebnis | Punkte |
|----------|--------|
| Genaues Ergebnis getippt (z. B. 2:1 = 2:1) | **3 Punkte** |
| Richtige Tordifferenz (z. B. 3:2 statt 2:1) | **2 Punkte** |
| Richtige Tendenz (Sieg / Unentschieden / Niederlage) | **1 Punkt** |
| Weltmeister richtig getippt | **+5 Punkte** |

---

## 6. Häufige Fehler vermeiden

| ❌ Falsch | ✅ Richtig |
|-----------|-----------|
| Kinder tragen Ergebnisse selbst ein | Ergebnisse kommen **automatisch** — Schüler tippen nur ihre **Vorhersage** |
| Code vergessen → neues Kind anlegen | Im Admin-Bereich: **Code zurücksetzen** (`#Name` suchen) |
| Zwei Kinder teilen sich einen Code | Jedes Kind hat seinen **eigenen Code** |
| Echten Namen als Phantasienamen eingeben | Immer **erfundene** Namen verwenden |

---

## 7. Technik kurz checken

- Alle Geräte müssen im **gleichen Schul-WLAN** sein.
- App-Adresse: `http://<IP-des-Schulrechners>:3003`
  - IP herausfinden: im Terminal `ipconfig getifaddr en0` (Mac) eingeben.
- Die Adresse oder den QR-Code an der Tafel zeigen, damit alle Kinder die App öffnen können.
